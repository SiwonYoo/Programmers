# Programmers
이 레포지토리는 프로그래머스(Programmers) 문제 풀이를 정리한 저장소입니다.

주 언어는 JavaScript이며, JS 지원이 안 되는 경우 C++을 사용합니다.

## 문제 풀이 규칙
- 첫 성공 전까지 AI 사용이나 풀이 검색 금지 (Mdn만 참고 가능)
- 문제 풀이에 너무 많은 시간이 소요될 경우, 프로그래머스 질문에서 테스트 케이스만 참고
- 문제 성공 이후, 다른 사람 풀이를 참고하거나 AI를 통해 개선할 점 찾아보고 적용해서 올리기
  - 맨 윗줄에 주석으로 개선점 작성하고, 설명이 필요한 부분 코멘트 달기
